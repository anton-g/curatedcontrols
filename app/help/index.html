Help

Curated Controls is a collection of hand-picked open source iOS components. All included components is required to be of high quality.

New controls are added every day and we appreciate a tip if you know a great control we're not listing.

A lot of the controls are available through the dependency manager CocoaPods. If you haven't used CocoaPods you can read more about it here and how to get started here.

Beta

We are currently in beta and the site is still under development. If you find any errors, bugs or have any feedback please let us know. You can send an email to hello@curatedcontrols.com or just reach us at @curatedcontrols.
